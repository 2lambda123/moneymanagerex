Бюджет

MMEX позволяет настроить бюджет на год и/или месяц.

Это позволит вам сравнить исполнение бюджета с его планированием.

Бюджетные планы могут отображаться в календарных или финансовых годах.

При необходимости, используйте диалоговое окно «Параметры», чтобы установить начало финансового года и любые постоянные настройки.

Опция Вид→Бэджет: Как финансовый год также может быть использована для переключения между двумя разными представлениями бэджета.

Создание нового бюджета

Чтобы настроить планирование бюджета, щелкните правой кнопкой мыши узел дерева «Бюджетирование» и добавьте в бюджетный план месяц или год.

Редактирование категорий бюджета

После того, как год будет добавлен, выберите его в навигаторе для настройки.

Затем каждая категория может быть отредактирована двойным кликом мыши.

Это делается для всех последующих категорий.

Настройка бюджета

Это становится бюджетом на этот год. Последующие месяцы и/или годы могут быть скопированы из этого года.

Сводка по категориям отображается для каждой основной категории.

При настройке бюджета, итоговые суммы можно отключить, активировав пункт меню:

Вид→Бюджет: без подитогов

Используя отчеты в разделе "Бюджет", теперь можно сравнить, как вы фактически потратили деньги в сравнении со значениями вашего бюджета.
